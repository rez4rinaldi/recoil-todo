## To get started

1. Clone the repo

```bash
git clone https://github.com/jacques-blom/recoil-todo-list.git
```

2. Install dependencies using Yarn or npm

```bash
cd recoil-todo-list
yarn            #using yarn
npm install     #using npm
```

3. Start the development server

```bash
yarn start      #using yarn
npm start       #using npm
```
